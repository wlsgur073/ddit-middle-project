해당 페이지에서 재직 증명서를 신청하시면 관리자의 검토 이후 재직회원 권한을 부여합니다. 자세한 내용은 아래의 이미지를 참고해 주세요.

    재직증명을 신청한 회원을
    관리자가 조회

  2. 승인 확인
    관리자는 재직증명서를 검토 후
    회원에게 승인처리

  3. 결과 조회
    승인 처리된 회원은
